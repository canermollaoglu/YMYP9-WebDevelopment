CSS - Cascading Style Sheets

Tanımla: Web sayfalarının görünümünü ve düzenini belirlemek için kullanılan bir stil dilidir. HTML ile birlikte kullanılır ve web sayfalarının tasarımını kontrol etmek için kullanılır.

CSS, HTML sayfalarındaki metinlerin, resimlerin, bağlantıların, tabloların, listelerin ve diğer HTML öğelerinin nasıl görüneceğini belirlemek için kullanılır.

Bir parent öğeye herhangi bir css uygularsak bu içerisinde bulunan tüm çocuk öğelere de uygulanacağı anlamına gelir.

Bir html sayfa yapısına css eklemenin 3 yolu vardır:

  • Inline CSS:Html etiketlerinin içerisine yazdığımız bir yöntemdir.
  • Internal CSS: Sayfanın head kısmında style etiketi içerisine doğrudan css kodu ekleyerek yapılır.
  • External CSS: Css dosyası oluşturularak sayfanın head kısmına link etiketi kullanılarak eklenir.
